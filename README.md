# 🤖 Hadi AI - مرشدك الذكي لإتقان الإنجليزية

Hadi is an AI-powered English tutor built specifically for Arabic speakers.

## 🎯 الميزات الحالية
- ✅ **المدقق الذكي:** يصحح الأخطاء القواعدية والإملائية.
- ✅ **شرح بالعربية:** يشرح القاعدة بلغة عربية واضحة.
- 🔜 **مرشد المحادثة:** التدرب على التحدث (قيد التطوير).

## 🛠️ التقنيات المستخدمة
- Python
- OpenAI API (GPT-4o-mini)
- Google Colab
